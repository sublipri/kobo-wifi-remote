If you run into any problems, follow these steps:

1. Reboot your e-reader. (Fully power it off then turn it on again.)
2. From the initial setup page, redo the input check and re-record the page turning actions.
3. Reproduce the problem you were having. If it involves a custom action, re-record the action first.
4. If possible, try steps 2 & 3 with a different web browser and see if anything changes.
5. Download a log file using the button below.
6. If you require help, either post on MobileRead or open an issue on GitHub. Describe your problem and attach the log file. State your e-reader model and firmware version, web browser, and operating system. Attach a screenshot if you think it would be useful.
