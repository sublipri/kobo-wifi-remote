Trackpad mode active. Move your finger across the touchscreen to control the cursor, then tap to perform input.

Single tap = short press
Double tap = long press
Triple tap = exit trackpad mode

Arbitrary input mode active. Use the mouse or arrow keys or WASD to control the cursor on your e-reader. Hold Control and/or Shift to adjust the arrow keys' move distance.

Space or Mouse1 = perform input (hold for long press)
C = toggle mouse tracking (so you can re-position the mouse)
Q = exit arbitrary input mode
R = Re-initialize (use if the e-reader changes rotation)
